## 说明

每个技能都放在单独的文件夹中，包含详细的说明文档。当需要执行任务时，应先查看此仓库中是否有相关的技能可以参考和使用。

## 技能管理机制

1. **创建新技能**：当遇到新问题并成功解决后，应创建新的技能文档
2. **更新现有技能**：当现有问题有新进展或发现新细节时，应及时更新对应技能文档
3. **定期回顾**：定期检查和整理已有技能，确保内容准确有效
4. **技能复用**：在面对新问题时，优先查找已有技能，避免重复解决问题

## 添加新技能

要添加新技能，请创建一个新的文件夹，命名格式为`[技能名称]`，并在其中添加相应的说明文档。可参考[sill-template.md](./skill-template.md)。
